## 課題
下記の関数を書いてください。

> 長さ 3 の文字列 S が引数として与えられます。
> S に 1 度だけ含まれる文字を 1 つを返り値にする関数を書いてください。
> 但し、そのような文字が存在しない場合は代わりに -1 を返り値にしてください。


- 例1：aab -> b
- 例2：abc -> a または b または c(※ どれでもいいので)
- 例3：ppp -> -1 (1度だけ含まれる文字がないので)
